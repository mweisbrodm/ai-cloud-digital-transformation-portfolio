# Azure Cloud Data Pipeline

## Executive Summary
This project demonstrates how to build a cloud-based data pipeline using Azure Virtual Machines, MySQL, and Blob Storage. It shows how organizations can modernize data processing and storage using scalable cloud infrastructure.

## Technical Summary
Azure virtual machines, MySQL database services, Blob Storage, Python automation, cloud architecture.

## Approach
- Configure Azure Virtual Machine
- Set up MySQL database
- Connect VM to database
- Build Python automation script
- Store output in Azure Blob Storage
- Visualize pipeline architecture

## Business Impact
Supports digital transformation by enabling scalable, secure, cloud-based data processing. Reduces infrastructure overhead and improves operational efficiency.
